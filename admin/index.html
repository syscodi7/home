Seguridad del panel
Las credenciales se guardan en el localStorage del navegador. Para mayor seguridad en producción, implementa autenticación en el servidor.
Guía de publicación
Con GitHub Pages (recomendado):
1. Configura tu token PAT en la seccion de arriba
2. Edita el contenido y haz clic en Guardar cambios
3. ✓ El archivo se actualiza en tu repo automaticamente
Con hosting PHP (cPanel, Hostinger):
1. Edita el contenido desde este panel
2. Haz clic en Guardar cambios
3. ✓ El sitio se actualiza automaticamente al instante
⚠ Para cambiar la contrasena de la API PHP, edita ADMIN_SECRET en admin/guardar.php
